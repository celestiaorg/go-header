Golang implementation of Celestia's data availability node types (`light` | `full` | `bridge`).

### Minimum requirements

| Requirement | Notes            |
|-------------|------------------|
| Go version  | 1.17 or higher   |

### Installation

```sh
git clone https://github.com/celestiaorg/celestia-node.git
make install
```
